详细描述

向服务器发送请求的基类。

客户端向服务器发送各种请求，需要完成各种功能。
请求将数据与底层的套接字分开，对套接字进行了一定的封装。
JRequestBase封装了一些各种请求都需要用到的功能。

连接到地址为address 端口为port 的服务器。
若套接字为空，则发射 error() 信号、从 QDebug 输出错误提示并立即返回。
