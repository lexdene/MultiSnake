详细描述

SubServer向MainServer发送相关信息。

SubServer启动后，需要向MainServer发送：

  1. SubServer相关信息。
  2. 游戏相关信息。
  3. SubServer与游戏的关联。

请求的过程：

  1. 通过 JRequestPort 获得用户信息服务的端口。
  2. 调用 connectToHost() 函数连接服务器。
  3. 调用 sendLoginHashCode() 函数发送Login hash code。
  4. 调用 send***() 发送相关信息。
