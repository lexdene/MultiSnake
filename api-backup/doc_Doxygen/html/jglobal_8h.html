类型定义文档

typedef qint16 JCode

整个项目通用的函数返回值类型。

一些函数的返回值代表函数是否执行成功。
0代表成功，其它的代表各种错误。

typedef qint32 JID

整个项目通用的一个ID类型。

整个项目中有很多地方需要用到ID，比如用户ID，服务器ID，游戏ID等。
在此统一将所有的ID定义为32位有符号整数。
