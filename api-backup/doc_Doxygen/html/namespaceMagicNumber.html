详细描述

定义网络通信的“魔数”。

网络通信需要确定是不是同一个协议。
在每个数据包的开头是魔数和数据长度，之后才是数据的内容。

参见:
JSocketBase

类型定义文档

typedef qint32 MagicNumber::JMagicNumber

“魔数”的定义。

统一网络通信格式，将魔数定义为32位带符号整数。
